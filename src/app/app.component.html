Little package to easily select multiple choices of a big set of data. It's perfect for manage ManyToMany relationships in your admin panel for example.
Demo

It's easy to see the result in realtime with the variable we bind to the component:

Number of choices: {{ selectedChoices.length }}
Last choice: {{ selectedChoices[selectedChoices.length - 1].name }} None

With datetime selector

If you use the `typeOfSearch='datetime'` you will have a datetime picker in place of the searchbar.

Value emmited by the search bar: {{ demo1 }}

With datetime range selector

If you use the `typeOfSearch='datetime-range'` you will have two datetime picker in place of the searchbar.
